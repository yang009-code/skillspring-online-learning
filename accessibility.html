Accessibility Statement

The pages use headings, labels for form fields, image alternative text, and links that can be reached with a keyboard. Tables can scroll on a narrow screen.

The navigation changes to a menu button on smaller screens. The three templates also give users different colour and layout choices.

This is still a student project, so it may not cover every accessibility need. A user can report a problem through the Contact page.
