SKILLSPRING ONLINE LEARNING

Live website:
https://yang1cz.myweb.cs.uwindsor.ca/skillspring/

Basic setup:
1. Put the skillspring folder inside public_html.
2. Create a MySQL database and user in DirectAdmin.
3. Import sql/database.sql in phpMyAdmin.
4. Enter the database values in includes/config.php on the server.
5. Test the website and administrator pages.

Sample accounts after the SQL import:
Admin: admin / admin123
Student: student / student123

Change the sample passwords before final submission. Add your GitHub address and complete docs/MEDIA_SOURCES.md.
