Refund Policy

SkillSpring does not process real payments, so it does not issue real refunds. The checkout page only demonstrates how an order can be stored in a database.

The administrator can change an example order to Cancelled or Refunded. This shows the order-management part of the assignment.

A user may send a message from the Contact page when an order needs to be reviewed. A real business would need a more detailed refund deadline and payment procedure.
