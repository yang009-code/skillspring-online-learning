# SkillSpring Online Learning

SkillSpring is a small online course website made with HTML, CSS, JavaScript, PHP, and MySQL. It is a classroom project and does not process a real payment.

The database starts with 20 courses. Each course has an access option and a support option. A visitor can browse and search the catalogue. A registered user can add courses to a session cart, complete the example checkout, open enrolled lessons, save progress, and leave a review.

The administrator area has pages for courses, users, orders, customer messages, reports, templates, and system checks.

## Main folders

- `admin` contains the administrator pages.
- `css` contains the shared stylesheet and three templates.
- `docs` contains the project instructions.
- `help` contains the help wiki.
- `images` contains the logo, learning map, and course covers.
- `includes` contains the shared PHP files.
- `js` contains the external JavaScript files.
- `media` contains the three course preview videos.
- `sql` contains the database setup file.
