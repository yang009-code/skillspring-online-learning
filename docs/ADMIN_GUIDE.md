# Administrator Guide

Log in with an administrator account and choose Admin from the main navigation.

## Courses

The Courses page lists all course records. Add Course inserts a new course and six standard options. Edit changes the course information. Deactivate changes the status to inactive instead of removing old order data.

## Users

The Users page shows registered accounts. The administrator can change an account between user and admin, and can change the status between active and disabled.

## Orders

The Orders page lists the example orders. The status can be changed to Pending, Completed, Cancelled, or Refunded.

## Messages

The Messages page shows questions submitted through Contact. The administrator can type a response and save it. The message status then changes to Answered.

## Reports, templates, and monitor

Reports draws an enrollment chart with the HTML canvas element and JavaScript. Themes changes the site-wide template between Light, Dark, and Blue. Monitor checks the PHP session, database connection, images, videos, CSS, JavaScript, and help files.
