# Work and Submission Checklist

Before submitting, go through the project and make sure you understand the pages you are handing in.

- Put your name and course information in the file comments where required.
- Read the PHP, JavaScript, CSS, and SQL and be ready to explain it.
- Rewrite any course text that you want to personalize.
- Finish the exact video source details in `MEDIA_SOURCES.md`.
- Add the real GitHub repository address to `README.md`.
- Keep the real database password out of GitHub.
- Test the Light, Dark, and Blue templates.
- Test the website on a desktop browser and a phone-sized browser window.
- Test registration, login, cart, checkout, lessons, progress, reviews, and contact messages.
- Test course, user, order, message, theme, report, and monitor pages as administrator.
- Follow the instructor's rules about outside help and acknowledgements.
