# User Guide

## Browsing courses

Open Courses from the main navigation. The catalogue displays the active course records from MySQL. Search can look for a title, category, instructor, or word in the description. Categories gives another way to browse.

## Creating an account

Open Register and enter a name, email, username, and password. The password needs at least six characters. After registration, open Login and use the new account.

## Buying a course

Open a course detail page. Choose one access plan and one support plan. JavaScript shows an estimated price. Add the course to the cart. The cart lets the user change options or remove a course.

Checkout requires a login. It asks for a name, email, and demonstration payment choice. It does not ask for a real card number. After checkout, the order and enrollment are stored in MySQL.

## Learning and progress

Open My Courses after checkout. Choose Open Lessons. A lesson can be marked complete, and the progress is stored in the progress table. The My Courses page calculates a completion percentage.

## Other user pages

Profile shows the account information and has an edit form. Order History shows earlier orders. A user who bought a course can leave a rating and comment. Contact sends a question to the administrator.
